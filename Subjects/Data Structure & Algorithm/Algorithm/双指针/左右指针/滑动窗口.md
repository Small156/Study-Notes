# 滑动窗口

滑动窗口算法能在给定的特定窗口大小的数组或字符串上执行操作，而不在整个字符串和数组上。故该算法可以降低循环的嵌套深度，以减少时间复杂度。

滑动窗口主要用于解决：数组 / 字符串的 **子元素** 问题。
